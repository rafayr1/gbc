# COMP1238 - Course Notes

## Week 1 Historical Introduction to Text and Writing
- Writing originated 5000 years ago in Mesopotamia using cuneiform for administrative records
- Babylonians used a base 60 positional system which influenced modern time measurements

## Week 2 Bits and Bytes
- RAM serves as volatile memory for active tasks while SSDs provide non volatile storage
- A byte consists of 8 bits representing values from 0 to 255 in an unsigned system

## Week 3 Structure of Text
- Markdown provides a human readable syntax while HTML uses a verbose tag based structure
- UTF 8 is the standard internet encoding and remains backwards compatible with ASCII

## Week 4 Keyboards and Editors
- Developers use text editors to avoid the hidden formatting metadata found in word processors
- Regular expressions allow for advanced pattern matching such as finding digits with regex

## Week 5 Command Line Interfaces
- CLI is essential for remote server management and allows for task automation through scripts
- Basic navigation uses pwd to display the path and ls to list files in a directory
